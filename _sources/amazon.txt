==========================

.. note::
    美国亚马逊的商品最全了，学会淘美国亚马逊后，什么东西都可以海淘到了。

下单流程
------------------

下单图文流程:

1. 打开亚马逊的网址，www.amazon.com 点击页面右上角的Your Account后,会出来如下图页面，如果已经\
   有美国亚马逊帐号可以直接点Sign in按钮去登录，没有亚马逊帐号的就点击下图中被箭头指向的"Start here"处哦

.. image:: d6e3d8b6jw1ecuv4xbp66j20hl0augo4.jpg

2. 点击Start here 后就进入新用户注册流程，把相关信息填写好，具体如何填写的请看下图:请注意注册的邮箱和密\
   码都要输2遍哦！填好后点击Create account按钮就完成注册了！

.. image:: d6e3d8b6jw1ecuvef1hvbj20n10f8acw.jpg

3. 完成上一步操作之后，就会自动登录此帐户，会出现如下页面

.. image:: d6e3d8b6jw1ecc3ugjgyzj20h10aiabr.jpg

4. 下面就可以开始选择需要购买的东西了：可以通过关键词查找也可以通过类目查找，记得要用英文去搜，英文不好的\
   可以装个Google浏览器自动翻译或者直接去www.zhuzhuhaitao.com去用中文关键词模糊搜索。比如搜小鹿牙胶可以用\
   关键词搜，如下图：


.. image:: d6e3d8b6jw1ecc3x2hb5ij20md0dtq62.jpg

5. 出来搜索结果页面，会有给出很多的选项，都符合的话一般点最上面那个的标题进入就可以，如下图：

.. image:: d6e3d8b6jw1ecc3xnyvxgj20l50i177n.jpg

6. 进入商品详情页后是这样的页面，会有商品价格，描述等信息，这里看不懂英文介绍的话可以把商品链接复制\
   进 http://www.zhuzhuhaitao.com 去查看中文信息，重量、到手价、价格曲线等等都能查到哦，想购买的话点页面右边的加\
   入购物车按钮，如下图标注的：

.. image:: d6e3d8b6jw1ecc3z680psj20vr0cpwi0.jpg

7. 要买的东西都添加进购物车以后，可以按”proceed to checkout“按钮去结帐，如下图：

.. image:: d6e3d8b6jw1ecdi0wh1utj20tj0bs41x.jpg

   转运的请看这个转运攻略 http://bbs.zhuzhuhaitao.com/thread-37-1-1.html 。我以我用的腾牛速递为例给大家介绍下\
   怎么填，以下是腾牛速递给我的地址（注册过才有用哦）：

.. image:: d6e3d8b6jw1ecdi8p0613j20qr0ilq78.jpg

9. 然后把转运给的地址依样画葫芦的填进亚马逊的地址栏中，如下图:

.. image:: d6e3d8b6jw1ecdigte3unj20dn0jjgnp.jpg

10. 填完地址以后点ship to this address按钮，就会让你确认货物信息和选择发货速度，如下：

.. image:: d6e3d8b6jw1ecdid06pjoj20va0hujur.jpg

11. 只有选最上面的那个5到8个工作日的选项才是免美国运费的，如果嫌慢可以免费试用3个月的亚马逊妈妈计划，就\

查发货状态、快递单号
-------------------------------------

有些人反映买完商品后不知道如何去看商品的发货状态和快递单号，我以我下的一个单为例给大家讲解下！ 


1. 点开页面右上角的"Your Account",再点"Your Orders" 如下图标注：

.. image:: d6e3d8b6jw1ecuxbia8nlj20nx0dx41t.jpg

2. 点进去以后就是已买到的商品页面了，发货状态看下图所示：

.. image:: d6e3d8b6jw1ecuxe7k11hj20s00gc41y.jpg

3. 如果要查看美国快递单号等信息，请点击左边的Order Details按钮（如上图图上标注的）进入，进入后的页面如下图：

.. image:: d6e3d8b6jw1ecuxih5cszj20sr0gyae2.jpg

4. 看上图标注，美国发货的快递公司和快递单号都能在这个页面查到，有些转运公司会要求入库预报，就是把快递公司名和\
   快递单号去转运公司填写到对应页面就行。亚马逊是发货才扣款的，发货的时候也会有发货邮件，以前快递单号会写在邮件里，\
   现在基本不写邮件了，如果你选择的转运公司是需要填入库预报的话就到亚马逊的这个页面去查快递单号，\
   只要商品的状态显示Shipped或者Delivered就能查到单号的。





查询重量、到手价、价格曲线
-----------------------------------------

亚马逊商品的中文介绍，重量，到手价，价格曲线，判断自营或第三方等等信息都能通过猪猪海淘查到哦，还可以添加进购物车查看\
购物车的总重量呢，甚至可以支持模糊的中文搜索商品哦！具体教程请往下仔细看哦！ 

1. 首先打开猪猪海淘的网址：http://www.zhuzhuhaitao.com （请大家加入收藏夹，方便下次找到哦！）猪猪海淘首页是海淘过亚马逊\
   的人曾经买的东西，可以看看别人购买过的东西，有需要的话点入对应的商品图片可以进入到猪猪海淘的商品介绍页面，在商品介绍\
   页面可以直接加入购物车哦！

.. image:: d6e3d8b6jw1ecy5n6rmu4j20q80gdn0l.jpg

2. 点左边导航栏对应的分类进入，会显示亚马逊对应类目下热门的商品哦，而且每个商品都有中文标题，一目了然的知道是什么东西。

.. image:: d6e3d8b6jw1ecy5o37gmsj20ra0gntbp.jpg

3. 在猪猪海淘上大家可以找到亚马逊商品的中文介绍，重量，到手价，价格曲线，还可以区分自营和第三方产品哦！下面我们以亚马逊\
   的一款吸管杯为例，给大家介绍猪猪海淘的强大功能，其他的亚马逊商品依样画葫芦即可。下图为亚马逊商品详情页，选择好款式后，\
   复制商品链接，不知道商品链接是什么的看图片：

.. image:: d6e3d8b6jw1ecy5q6v6ilj20se0k2n2v.jpg

4. 把刚才复制的商品链接粘贴进搜索框：

.. image:: d6e3d8b6jw1ecy5qps1t5j20l40ekac5.jpg

5. 然后点搜索按钮，出来的就是如下页面： 看到没有？商品的中文介绍，价格，重量，自营还是第三方的判断（建议不买第三方产品），\
   到手价（加上预计的运费后的价格）应有尽有啊，有了这样的页面谁还能以看不懂英文作为逃避海淘的理由呢？

亚马逊妈妈计划免费试用教程
----------------------------------------

美国亚马逊妈妈计划是亚马逊针对客户体验做出的一项新的服务提升，收到北美数千万网购妈妈的喜爱，亚马逊妈妈计划的优势： 

* 买尿片和湿巾可以有8折优惠。 
* 免费试用3个月的prime计划，两日超快免费送货，普通帐号可是要一周左右哦，而且不满25美金也免运费 
* 其他超值折扣。


申请试用的教程：

已有美国亚马逊帐号的直接登录，没有美国亚马逊帐号的可以先到http://www.amazon.com  点击右上角your account，再点start here去注册帐号，如下页面，把注册邮箱和密码信息填好

.. image:: http://bbs.zhuzhuhaitao.com/data/attachment/forum/201311/04/200731awycztjjzg66zas6.jpg

注册好登录以后点击以下链接进入亚马逊妈妈的申请页面：
https://www.amazon.com/gp/mom/account?ie=UTF8&ref_=ya_manage_mom&tag=mmqqg-20

到达如下页面，点击“join now"进入：

.. image:: http://bbs.zhuzhuhaitao.com/data/attachment/forum/201311/04/200836ijr6errbnoorl8zo.jpg

进入如下页面，把对应信息填写好，具体内容见图，填写好点击continue按钮：

.. image:: http://bbs.zhuzhuhaitao.com/data/attachment/forum/201311/04/200932lxgo88odh3hoo8du.jpg

进入到如下页面，填写信用卡信息和地址，如果你账户有绑定过信用卡，可以选已经有的卡（existing card），如果没有则新建一个卡（选new card）。注意这里的地址是要填信用卡的账单地址，就是平常每个月信用卡的纸质账单要寄去哪，不知道的可以上信用卡的网上银行查询或拨打银行电话查询，写拼音就可以，邮编随意去除一位即可（因为美国邮编是5位，中国邮编是6位）。

.. image:: http://bbs.zhuzhuhaitao.com/data/attachment/forum/201311/04/201011iw99j8smrpfx9u9p.jpg

点击黄色按钮后就试用成功啦！加入成功就是这个页面啦！

.. image:: http://bbs.zhuzhuhaitao.com/data/attachment/forum/201311/04/201209i03bx40u0i0b5j53.jpg


取消续费
^^^^^^^^^^^^^^^^^^

在加入成功后记得要取消自动续费功能哦(在3个月到期之前任何一天取消都可以），不然三个月免费试用到期会自动扣费79美元哦（因为亚马逊妈妈的年费是79美元），下面介绍取消自动续费的方法：

右上角Your Account进入

.. image:: http://bbs.zhuzhuhaitao.com/data/attachment/forum/201311/04/201240ymk24k6mgd4ze2dc.jpg

进入到如下页面，再点Manage prime membership，因为这个自动续费实际上是prime的年费

.. image:: http://bbs.zhuzhuhaitao.com/data/attachment/forum/201311/04/201325ajkki2djscokcmk2.jpg

进入以后到达以下页面：

.. image:: http://bbs.zhuzhuhaitao.com/data/attachment/forum/201401/10/150109gkzigs6rsq6rcjic.jpg

点击End Membership按钮，然后会出现此页面，让你确定是否要退出

.. image:: http://bbs.zhuzhuhaitao.com/data/attachment/forum/201401/10/151048u7v5b95m5pqmsdxp.jpg

直接点End Membership按钮，也有可能改版后上面这个页面不出现，直接出现如下页面：

.. image:: http://bbs.zhuzhuhaitao.com/data/attachment/forum/201401/10/151224g2zei1f4al2ccflv.jpg

点End Now按钮就直接退出prime啦，退出成功的页面如下图：

.. image:: http://bbs.zhuzhuhaitao.com/data/attachment/forum/201401/10/151318zv6sot48aomyt8xq.jpg

.. note::
    只要在妈妈计划免费3个月到期之前去退出Prime就行，万一忘了退出3个月到期的时候被扣了79美元（有时候会扣80多美元），只要在扣费后没有购买过亚马逊的商品，都可以遵照我上面这个退出prime的步骤来操作，操作成功后邮箱马上会收到一封亚马逊发的邮件，邮件的内容就说没使用到Prime的好处，所以扣的款项会退回，然后自己不用操作什么，等着钱退回就行。

亚马逊联系客服的方法
-------------------------------------------

点此链接进入： https://www.amazon.com/gp/help/contact-us/general-questions.html?ie=UTF8&nodeId=508510&type=email&skip=true&ie=UTF8&tag=mmqqg-20&linkCode=ur2&camp=1789&creative=390957 打开后是如下页面：

退货也属于订单问题，所以第一项选”An order I placed”,选好后就会出现已买过的商品，在要退货的商品前打勾，然后把第二项的选项都选好，我选的是退货的选项，具体大家遇到了具体什么问题要联系客服的话根据自己遇到的情况勾选选项哦！如下图：

.. image:: http://ww2.sinaimg.cn/large/d6e3d8b6jw1ee7exuj1f8j20o60hptbs.jpg

.. image:: http://ww3.sinaimg.cn/large/d6e3d8b6jw1ee7f44xvdtj20oh0g6jte.jpg
